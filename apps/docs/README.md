# Metador

**Limits enforced by the chain, not by trust.**

Metador is a marketplace of non-custodial strategy vaults built on DeepBook, Sui's on-chain order book. Skilled traders open vaults and run strategies inside them. Savers deposit and copy those strategies — without handing custody of their funds to the trader.

The safety guarantee is structural, not contractual. Even if a vault operator's private key is stolen, depositor funds cannot be withdrawn and spending limits cannot be exceeded. These constraints are enforced by Sui validators, not by Metador's servers. There is no Metador-held key that could break them.

Metador is live on testnet. Mainnet launches under hard caps after external review (see [Roadmap](roadmap.md)).
